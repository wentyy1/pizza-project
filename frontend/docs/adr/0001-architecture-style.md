# ADR 0001: Вибір архітектурного стилю - Модульний моноліт

## Контекст

### Стан проєкту
- Проєкт "Pizza Day" знаходиться на початковій стадії розробки
- MVP системи замовлення піци з базовим функціоналом
- Доменна модель визначена: Меню, Замовлення, Користувачі, Доставка

### Команда
- Невелика команда (1 розробник)
- Обмежені ресурси для підтримки інфраструктури
- Невеликий досвід роботи з монолітними архітектурами

### Процеси
- Ітеративна розробка з короткими циклами
- Потреба швидкого виходу на ринок
- Висока частота змін на початковому етапі

## Рішення

**Обрано модульний моноліт** з чіткими внутрішніми межами між бізнес-контекстами.
